BOX의 기본 성질 (display: block;)

  • 투명하다.
  • width: auto / height: auto
  • Y축으로 쌓인다.
  • 자신의 요소 옆에 다른 요소가 오지 못한다.
  1. 태그는 크게 inline / block 속성으로 나뉜다.
    2. 글자 그림 등 내용에 관여하는 태그: span, strong, em, icon, ,
    인라인 속성을 가지고 나머지 태그들은 Block 속성을 가진다.
  2. Block 속성은 width height margin padding 값을 가지고 inline 속성은 margin,padding의 left/right값만을 가진다.
  3. inline 속성은, padding의 top/bottom을 설정하면 background를 포함하여 padding이 적용된 것처럼 보인다.
